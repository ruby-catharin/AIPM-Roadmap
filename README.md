# AI PM Mastery Roadmap

A structured, 4-week AI learning tool built specifically for Product Managers.

Unlike tutorials aimed at engineers or executives, this roadmap bridges the gap — helping PMs reason about token economics, RAG tradeoffs, agent reliability, and more. Every concept can be explored through four lenses:

- **ELI5** — for when you need the intuition before the vocabulary
- **Normal** — for when you are ready to understand how it actually works
- **Technical** — for when you want the papers, the math, the precise mechanics
- **PM Lens** — for when you need to translate the concept into a product, cost, or strategy decision

No completion badges, no progress bars — just the depth you need, when you need it.

## Stack

- React + Vite
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```
